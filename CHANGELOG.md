# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-14

### Added
- Initial release
- 11 theme variants (3 light, 8 dark)
- Full Treesitter support
- LSP semantic highlighting
- Support for popular plugins:
  - Telescope
  - NvimTree / Neo-tree
  - GitSigns
  - Indent Blankline
  - Lualine
  - Which-key
  - Alpha (dashboard)
  - Notify
- Complete port of Noctis VSCode theme colors
